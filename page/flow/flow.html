完整覆盖流程的 每一段。

02 / 04

动态关联 @erupt 类

已发起的流程数据 @Erupt 注解即表单 —— 字段类型、校验、关联查询全部沿用，无需重复定义。

@Erupt(name = "报销申请")
@Table(name = "reimburse")
public class Reimburse {
  // 流程引擎自动接管 ↓
  @EruptField(views = ...)
  private BigDecimal amount;
}

04 / 04

自定义 节点扩展 · 双重身份

Erupt 既是 流程表单引擎，也是 流程配置平台。继承 NodeHandler 实现自定义节点 —— 风控审核、合同盖章、企业微信审批，把组织里特有的环节也带进画布。

作为 表单引擎
复用 @Erupt 实体即可获得审批表单 + 数据持久化 + 审批历史
⇄
作为 配置平台
把现有业务流程接入引擎，沿用既有审批策略与组织架构
+ 通知

多通道 流程通知

邮件、企业微信、钉钉、飞书、站内信 —— 每个节点独立配置触达方式与模板。

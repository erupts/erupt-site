02 / 05
动态关联
@erupt 类
已发起的流程数据 @Erupt 注解即表单 —— 字段类型、校验、关联查询全部沿用，无需重复定义。
@Erupt(name = "报销申请")
@Table(name = "reimburse")
public class Reimburse {
  // 流程引擎自动接管 ↓
  @EruptField(views = ...)
  private BigDecimal amount;
}
